# README

This project is a Rails 5.2 app using webpacker and React.

### Ruby version

MRI 2.5.1

### Getting ready

- `bundle install`
- `yarn install`
- `cp .env.local.sample .env` -- fill in env vars where applicable

### Start it up

- `foreman start -f Procfile.development`
- http://localhost:5000
